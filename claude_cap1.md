# Claude Code 改変指示書：第一章「喪失と遺産」

この指示書に基づき、第一章を全面的にリライトしてください。また、舞台設定については外したくないポイントだけ記述しています。是非とも話の流れがきれいになるように、日常生活のいろいろな描写（講義・アルバイト・帰宅の道中・帰省の道中・・・その他いろいろ）を付け加えてください。

## 1. 舞台設定の変更（大学一年生へのシフト）
* **時期:** 大学一年生の5月〜6月。新生活の熱狂が一段落し、五月病の気配が漂う時期。
* **冒頭の追加:** * 大学構内（学食やラウンジ）の描写からスタートする。
    * 周囲がサークル仲間や「よっ友」のグループで固まり、賑やかに笑い合う中、湊だけがスマートフォンの画面を見つめて一人で食事をしている描写。
    * 4月の新歓期の無理なテンションが消え、自分だけが「居場所」を確保し損ねたという、新入生特有の取り残された感覚を強調する。

## 2. 帰省シーンの対話リライト
* **テーマの変更:** 話題を「就活」から**「大学生活と友人関係」**へ変更する。
* **母・真奈との会話:**
    * 母の問い：「友達、たくさんできた？」「サークルは入ったの？」
    * 湊の嘘：「……まあ、普通に。ゼミのやつらと飯食ったりしてるし。サークルは、まだ迷ってる」といった、実体のない返答。
    * 母の言葉：「大学の友達は一生ものだから。お母さんみたいに、仕事だけの人間にならないでね」という期待が、湊にとって呪いのように響く描写にする。

## 3. 伏線：母の夜職の伏線
* 母の死後、葬式での伯母との会話中、直接は言わないものの嫌みったらしく母が夜職をしていたことが示唆される。
    * 湊は最初は理解できない ものの、のちのちになって遺品整理の際に通帳を見てその意味を理解する。
    * 一方、救いがあるとすれば湊が生まれてからは多額の入金元が不明な振り込みはなかったことだった。それと同時に、2009年の8月、つまり湊が5歳の時から毎月20万円の送金があることを目にし、明確に両親の離婚した時期を把握する。湊は父の記憶が曖昧だった。

## 4. 喪失と絶望の再定義
* **遺産の意味:** * 3200万円の通帳を見つけた際の衝撃を、「これから4年間の大学生活を、誰とも繋がらず、働かなくても生きていけてしまう地獄」として描く。
    * 「自由」が、足場のない宙吊りのような恐怖に変わる瞬間を強調する。

## 5. テクニカルな修正（キーワード置換）
* **削除:** 「三年生」「就活」「OB訪問」「ゼミのレジュメ」「内定」などの単語をすべて削除。
* **追加:** 「新歓」「五月病」「必修科目の代返」「SNSのグループ」「ぼっち飯」など、一年生特有の孤独を表すワードを散りばめる。

## 6. 文体・トーンの指示
* 湊の一人称視点を維持。冷笑的だが、その裏にある「誰かに選ばれたい」という未熟な渇望を滲ませる。
* 18歳〜19歳の若者が直面する、重すぎる「金」と「孤独」の対比を鮮明に描くこと。
