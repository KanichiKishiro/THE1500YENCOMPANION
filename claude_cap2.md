# Claude Code 改変指示書：第二章「事業開始」

この指示書に基づき、第二章をリライト・増量し、湊が「金で買った日常」に耽溺し、摩耗していく過程を詳細に描写してください。是非とも話の流れがきれいになるように、日常生活のいろいろな描写（講義・アルバイト・帰宅の道中・帰省の道中・・・その他いろいろ）を付け加えてください。

## 1. 時間軸と動機の再定義（大学一年生・冬〜二年生・春）
* **時期:** 一年生の冬休み明けから、二年生に進級する時期。
* **動機:** 「このままあと三年、誰とも話さず卒業するのか」という恐怖。母の遺産を、皮肉にも「清らかな大学生の日常」を疑似体験するために投入するという背徳感を強調する。
* **「大学生らしいこと」への執着:** SNSで見かける「鍋パ」「深夜のドライブ」「テスト前の教え合い」といった光景への、執着に近い憧れを描写する。
* 雇用形態を具体化するまでの流れの具体化。（見よう見まねで作った公式SNSアカウントの作成）

## 2. 日常描写の具体化と「違和感」の増量
* **「線」としての描写:** 待ち合わせ、食事、解散という一連の流れを、より生活感のあるディテールで埋める。
    * 駅ビルの騒がしいチェーン店での注文のやり取り、会計時の「給与封筒」の受け渡しの生々しさ。
    * 解散した後に一人で帰る夜道の、急激な温度差と静寂。
* **「合わない」感覚の強調:** * 念願の「ファミレスでの数時間の駄弁り」を実行するが、湊だけが**「今、一分ごとに25円減っている（時給1500円÷60分）」**と脳内で計算してしまい、純粋に楽しめないノイズを描く。
    * 相手の笑い声が「演出」に見えてしまい、白けてしまう湊の冷めた視点。

## 3. 雇用相手とのエピソード深掘りと「継続」の伏線
各登場人物との関わりを、単なる「面接」ではなく「歪な日常の積み重ね」として描写してください。

* **大前連（工学部の猫背）:**
    * 160cm程度、眼鏡、マッシュヘア。
    * 専門用語をまくしたてる彼に、湊が「それってどういう意味？」と聞く。一見、友人同士の会話に見えるが、実は「聞き役という業務」を湊が発注している構造。
* **佐々木莉乃（インスタ女子）:**
    * 人生を楽しんでいそうなキャピキャピ系の女子。
    * 彼女の「映え」のために、湊が不自然な角度でコーヒーカップを持たされる等の具体的動作。彼女がスマホ越しにしか湊を見ないことへの、物質的な虚しさを強調。
* **工藤健治（体育会系元部長）:**
    * いかにも体育会という雰囲気。話のネタと言えば女、酒、タバコ。
    * 彼に誘われて行く「不慣れなスポーツ」や「居酒屋でのコール」。湊が無理をして「大学生のノリ」を演じるが、身体が拒絶する感覚。
* **内田弘樹:**
    * 最も長く続いた彼との、あまりに「事務的で完璧な」友情。
    * 授業の代返、過去問の共有、ノートのコピーなど、実利的な「友人代行」が完璧であればあるほど、湊の心が削られていく過程を丁寧に描く。
    * **※彼はこの章で契約終了せず、後の章（就活期や卒業後）でも有益な情報提供や、湊が就活を意識する契機となる。この章ではそれについて言及する必要はないが、継続的に出現する人物として、執筆における導線を残しておく。**

## 4. 感情の累積（結び）
* 二年生の春、桜が舞う中で「何人雇っても、自分のスマートフォンの通知は相変わらず『公式アカウント』か『父』だけである」という事実に直面させる。
* 牛丼屋での「タレを渡す」シーンを、**「無償の、しかし一瞬で消える関係」**の尊さと切なさとして描き直す。

## 6. 備考
* **描写** 日常描写をもっと付け加えてほしい。特に上に上げた雇用者とのやりとりについてはどんどん深掘りたい。例えば同じ授業に出席する、テーマパークやスポットなどへのお出かけ等。それぞれの登場人物との関係性の変化についてはもっと具体的に記述してよい。
また、同じ時期に一人しか雇用しないわけでもない。
